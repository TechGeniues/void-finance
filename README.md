# VOID — Personal Finance Tracker

A clean, minimal personal finance tracker and budget manager built with vanilla HTML, CSS, and JavaScript.

**Live Demo:** https://TechGeniues.github.io/void-finance

---

## Features

- **Dashboard** — Balance overview, income vs expenses, savings rate
- **Budget Manager** — Set monthly limits per category with visual progress bars
- **Transaction Tracker** — Log income and expenses, filter by type and category
- **Spending Breakdown** — Visual category chart for monthly expenses
- **Persistent Storage** — All data saved locally in your browser via localStorage
- **Zero Dependencies** — Pure HTML/CSS/JS, no frameworks

## Tech Stack

- HTML5
- CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- localStorage for data persistence

## Getting Started

```bash
git clone https://github.com/TechGeniues/void-finance.git
cd void-finance
# Open index.html in your browser — no build step needed
```

Or just visit the live demo link above.

## Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://TechGeniues.github.io/void-finance`
