# VOID — Finance OS

> Dark. Minimal. Functional.

A personal finance tracker and budget manager built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies, no build step.

**Live Demo:** https://TechGeniues.github.io/void-finance

---

## Features

- **Dashboard** — KPI cards, spending breakdown, budget health at a glance
- **Transactions** — Full log with filtering by type and category
- **Budget Manager** — Set monthly limits with animated progress indicators
- **Add Entry** — Quick-add income or expenses with live feedback
- **Persistent** — Everything saves to localStorage
- **Responsive** — Works on mobile with collapsible sidebar

## Stack

- HTML5 / CSS3 / Vanilla JS (ES6+)
- Google Fonts (Space Mono + Syne)
- localStorage for persistence
- Zero npm packages, zero build tools

## Run Locally

```bash
git clone https://github.com/TechGeniues/void-finance.git
cd void-finance
open index.html
```

## Deploy to GitHub Pages

1. Push repo to GitHub (public)
2. Settings → Pages → Source: main / root
3. Live at `https://[YOUR-USERNAME].github.io/void-finance`
